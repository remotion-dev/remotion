# Remotion plugin for Kimi Code

Create programmatic videos with Remotion and React using Kimi Code. The plugin
includes Remotion's Agent Skills for animations, audio, captions, rendering,
media processing, and more.

See the [Kimi Code plugin documentation](https://www.remotion.dev/docs/ai/kimi-code-plugin).

## Installation

In Kimi Code, install the plugin from GitHub:

```text
/plugins install https://github.com/remotion-dev/kimi-code-plugin
```

Confirm that you trust the third-party plugin, then run `/new` or `/reload` to
activate it.

## Usage

Ask Kimi Code to create or edit a Remotion video. Kimi Code automatically loads
the relevant skills, or you can invoke the main skill directly:

```text
/skill:remotion-best-practices
```
