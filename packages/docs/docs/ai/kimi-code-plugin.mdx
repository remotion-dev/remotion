Remotion maintains a [plugin for Kimi Code](https://github.com/remotion-dev/kimi-code-plugin).

It can help you create and edit Remotion projects and includes [Remotion Agent Skills](/docs/ai/skills).

## Installation

In Kimi Code, install the plugin from GitHub:

```text title="Kimi Code"
/plugins install https://github.com/remotion-dev/kimi-code-plugin
```

Confirm that you trust the third-party plugin. Run `/new` or `/reload` to activate it.

## Usage

Ask Kimi Code to create or edit a Remotion video. Kimi Code automatically loads the relevant skills.

You can also invoke the main skill directly:

```text title="Kimi Code"
/skill:remotion-best-practices
```
