# makeDragPreviewMimeType()<AvailableFrom v="4.0.498" />

Constructs a versioned MIME type containing small, non-sensitive metadata for a drag preview.

```ts twoslash title="drag.ts"
import {makeDragPreviewMimeType} from '@remotion/drag-and-drop';

const mimeType = makeDragPreviewMimeType({
  kind: 'composition',
  width: 1920,
  height: 1080,
  durationInFrames: 150,
});
// application/vnd.remotion.preview;v=1;kind=composition;width=1920;height=1080;duration=150
```

Use the result as the format passed to `DataTransfer.setData()`.
[`setDragPreviewMetadata()`](/docs/drag-and-drop/set-drag-preview-metadata) performs both steps.

## Arguments

### kind

`"asset"`, `"composition"`, or `"element"`.

### width?

The width in pixels. If set, `height` must also be set. Default: omitted.

### height?

The height in pixels. If set, `width` must also be set. Default: omitted.

### durationInFrames?

Available for compositions and Elements. The duration in frames. Default: omitted.

### durationInSeconds?

Available for assets. The duration in seconds. Default: omitted.

The function throws if a value is invalid or outside the supported bounds.
