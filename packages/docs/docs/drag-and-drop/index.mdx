## MIME types

Each payload family has an exported MIME type:

- `ASSET_DRAG_MIME_TYPE`
- `COMPONENT_DRAG_MIME_TYPE`
- `COMPOSITION_DRAG_MIME_TYPE`
- `EFFECT_DRAG_MIME_TYPE`
- `ELEMENT_DRAG_MIME_TYPE`
- `SFX_DRAG_MIME_TYPE`

Use the matching MIME type as the first argument to `DataTransfer.setData()`.
