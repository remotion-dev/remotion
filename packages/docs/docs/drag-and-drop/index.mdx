## MIME types

Each payload family has an exported MIME type:

- `ASSET_DRAG_MIME_TYPE`
- `COMPONENT_DRAG_MIME_TYPE`
- `COMPOSITION_DRAG_MIME_TYPE`
- `EFFECT_DRAG_MIME_TYPE`
- `ELEMENT_DRAG_MIME_TYPE`
- `SFX_DRAG_MIME_TYPE`

Use the matching MIME type as the first argument to `DataTransfer.setData()`.

## Drag previews

[`setDragPreviewMetadata()`](/docs/drag-and-drop/set-drag-preview-metadata) stores small, non-sensitive metadata in a versioned MIME type.
Unlike the payload, this metadata can be read from `DataTransfer.types` while handling `dragover`.

Use [`getDragPreviewMetadata()`](/docs/drag-and-drop/get-drag-preview-metadata) to parse it and render placement feedback before the item is dropped.
The regular JSON payload remains the canonical source of data after the drop.
