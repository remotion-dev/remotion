# parseSfxDragData()<AvailableFrom v="4.0.498" />

Parses and validates a serialized sound effect payload.

```ts twoslash title="drop.ts"
import {SFX_DRAG_MIME_TYPE, parseSfxDragData} from '@remotion/drag-and-drop';

declare const dataTransfer: DataTransfer;
const payload = parseSfxDragData(dataTransfer.getData(SFX_DRAG_MIME_TYPE));
```

Returns `SfxDragData` when valid and `null` otherwise.
