# parseElementDragData()<AvailableFrom v="4.0.498" />

Parses and validates a serialized Remotion Element payload, including its source code, dependencies, slug, and dimensions.

```ts twoslash title="drop.ts"
import {ELEMENT_DRAG_MIME_TYPE, parseElementDragData} from '@remotion/drag-and-drop';

declare const dataTransfer: DataTransfer;
const payload = parseElementDragData(dataTransfer.getData(ELEMENT_DRAG_MIME_TYPE));
```

Returns `ElementDragData` when valid and `null` otherwise.
