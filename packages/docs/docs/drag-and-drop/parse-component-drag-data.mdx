# parseComponentDragData()<AvailableFrom v="4.0.498" />

Parses and validates a serialized component payload.

```ts twoslash title="drop.ts"
import {COMPONENT_DRAG_MIME_TYPE, parseComponentDragData} from '@remotion/drag-and-drop';

declare const dataTransfer: DataTransfer;
const payload = parseComponentDragData(
  dataTransfer.getData(COMPONENT_DRAG_MIME_TYPE),
);
```

Returns `ComponentDragData` when valid and `null` otherwise.
