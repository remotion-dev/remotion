# parseDragPreviewMimeType()<AvailableFrom v="4.0.498" />

Parses one MIME type created by [`makeDragPreviewMimeType()`](/docs/drag-and-drop/make-drag-preview-mime-type).

```ts twoslash title="drag.ts"
import {parseDragPreviewMimeType} from '@remotion/drag-and-drop';

const metadata = parseDragPreviewMimeType(
  'application/vnd.remotion.preview;v=1;kind=composition;width=1920;height=1080;duration=150',
);
```

Returns `DragPreviewMetadata` if the MIME type is valid, otherwise `null`.
Unknown versions, unknown fields, duplicate fields, and values outside the supported bounds are rejected.

Use [`getDragPreviewMetadata()`](/docs/drag-and-drop/get-drag-preview-metadata) to search all MIME types on a drag.
