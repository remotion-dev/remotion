# &lt;Solid&gt;<AvailableFrom v="4.0.464" />

Renders a solid-color rectangle on a `<canvas>` element.  
Use it for colored backgrounds or overlays in your composition.

```tsx twoslash title="MyComp.tsx"
import {AbsoluteFill, Solid} from 'remotion';

export const MyComp: React.FC = () => {
  return (
    <AbsoluteFill>
      <Solid color="#3344ff" width={1920} height={1080} />
    </AbsoluteFill>
  );
};
```

## API

### `width`

Width of the canvas in pixels. Must be a positive integer.

### `height`

Height of the canvas in pixels. Must be a positive integer.

### `color?`

Fill color of the rectangle.  
If omitted, the canvas stays transparent.

```tsx twoslash title="Transparent background"
import {AbsoluteFill, Solid} from 'remotion';

export const MyComp: React.FC = () => {
  return (
    <AbsoluteFill>
      <Solid width={1920} height={1080} />
    </AbsoluteFill>
  );
};
```

### `className?`

A class name to apply to the `<canvas>` element.

### `style?`

Inline styles to apply to the `<canvas>` element.  
The component also supports the visual transform fields documented on [`<Sequence>`](/docs/sequence#style) (`style.translate`, `style.scale`, `style.rotate`).

## Sequence props

`<Solid>` is wrapped in a [`<Sequence>`](/docs/sequence) with `layout="none"`.  
You can pass [`from`](/docs/sequence#from), [`durationInFrames`](/docs/sequence#durationinframes), [`name`](/docs/sequence#name) and other sequence props to time-shift the layer.

## Adding a ref

You can add a [React ref](https://react.dev/learn/manipulating-the-dom-with-refs) to a `<Solid>` component. If you use TypeScript, type it with `HTMLCanvasElement`:

```tsx twoslash title="MyComp.tsx"
import {useRef} from 'react';
import {Solid} from 'remotion';

const MyComp = () => {
  const ref = useRef<HTMLCanvasElement>(null);
  return <Solid ref={ref} color="red" width={100} height={100} />;
};
```
