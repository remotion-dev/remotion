# `blur()`<AvailableFrom v="4.0.464" />

Applies a separable Gaussian blur to a canvas-backed component using WebGL2.

## Example

```tsx twoslash title="MyComp.tsx"
import {AbsoluteFill, Solid} from 'remotion';
import {blur} from '@remotion/effects/blur';

export const MyComp: React.FC = () => {
  return (
    <AbsoluteFill>
      <Solid
        color="#3344ff"
        width={1920}
        height={1080}
        effects={[blur({radius: 8})]}
      />
    </AbsoluteFill>
  );
};
```

You can also import from the package root:

```tsx
import {blur} from '@remotion/effects';
```

## API

Pass an object with the following properties.

### `radius`

The blur radius in pixels. Must be a finite number.

### `horizontal?`

Whether to blur along the horizontal axis. Defaults to `true`.

### `vertical?`

Whether to blur along the vertical axis. Defaults to `true`.

Set one axis to `false` for a directional blur:

```tsx
effects={[blur({radius: 8, horizontal: true, vertical: false})]}
```

### `disabled?`

When `true`, the effect is skipped. Defaults to `false`. This field is added automatically for all effects and can be toggled in the Remotion Studio timeline.

## Usage

Add the effect to the `effects` prop of a component that supports canvas effects, for example [`<Solid>`](/docs/solid), [`<Img>`](/docs/img), [`<Video>`](/docs/video), or [`<HtmlInCanvas>`](/docs/remotion/html-in-canvas).

```tsx
<Solid
  color="red"
  width={400}
  height={400}
  effects={[blur({radius: 12}), blur({radius: 4, horizontal: false})]}
/>
```

## Requirements

`blur()` uses a WebGL2 backend. During renders, enable WebGL via [`Config.setChromiumOpenGlRenderer()`](/docs/config#setchromiumopenglrenderer) (for example `'angle'`). See [Using WebGL during renders](/docs/remotion/html-in-canvas#using-webgl-during-renders).
