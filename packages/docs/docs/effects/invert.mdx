# invert()<AvailableFrom v="4.0.466" />

_Part of the [`@remotion/effects`](/docs/effects/api) package._

An invert effect that can be applied to canvas-based components such as [`<Video>`](/docs/media/video), [`<HtmlInCanvas>`](/docs/remotion/html-in-canvas) and [`<Solid>`](/docs/solid).

Use it for negative-color looks, x-ray style passes, or as a preprocessing step before stacking additional effects.

## Example

```tsx twoslash title="MyComp.tsx"
import {AbsoluteFill} from 'remotion';
import {Video} from '@remotion/media';
import {invert} from '@remotion/effects/invert';

export const MyComp: React.FC = () => {
  return (
    <AbsoluteFill>
      <Video src="https://remotion.media/video.mp4" effects={[invert({amount: 1})]} />
    </AbsoluteFill>
  );
};
```

## API

Pass an object with the following properties. You can also call `invert()` without arguments.

### `amount?`

Mix between original and inverted color. `0` leaves the layer unchanged, `1` makes it fully inverted. Defaults to `1`.

The alpha channel is preserved.

### `disabled?`

When `true`, the effect is skipped. Defaults to `false`.
