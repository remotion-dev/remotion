# colorKey()<AvailableFrom v="4.0.472" />

_Part of the [`@remotion/effects`](/docs/effects/api) package._

A classic color key effect that removes pixels close to a chosen key color. Use it on canvas-based components such as [`<Video>`](/docs/media/video), [`<HtmlInCanvas>`](/docs/remotion/html-in-canvas) and [`<Solid>`](/docs/solid).

## Example

```tsx twoslash title="MyComp.tsx"
import {AbsoluteFill} from 'remotion';
import {Video} from '@remotion/media';
import {colorKey} from '@remotion/effects/color-key';

export const MyComp: React.FC = () => {
  return (
    <AbsoluteFill>
      <Video
        src="https://remotion.media/video.mp4"
        effects={[
          colorKey({
            color: '#00ff00',
            threshold: 0.2,
            feather: 0.1,
          }),
        ]}
      />
    </AbsoluteFill>
  );
};
```

## API

Pass an object with the following properties. You can also call `colorKey()` without arguments.

### `color?`

The key color to remove. Accepts any CSS color string. Defaults to `'#00ff00'`.

### `threshold?`

Similarity threshold from `0` to `1`. Lower values key a narrower range of colors. Defaults to `0.2`.

### `feather?`

Edge softness from `0` to `1`. `0` gives a hard cutoff; higher values produce smoother transparency transitions. Defaults to `0.1`.

### `disabled?`

When `true`, the effect is skipped. Defaults to `false`.

## Requirements

`colorKey()` uses a WebGL2 backend. During renders, enable WebGL via [`Config.setChromiumOpenGlRenderer()`](/docs/config#setchromiumopenglrenderer) (for example `'angle'`). See [Using WebGL during renders](/docs/remotion/html-in-canvas#using-webgl-during-renders).
