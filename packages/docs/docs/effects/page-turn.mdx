# pageTurn()<AvailableFrom v="4.0.478" />

_Part of the [`@remotion/effects`](/docs/effects/api) package._

Turns a canvas-based component away like a curled page.

Use it for page flips, card exits, and book-style transitions on [`<Video>`](/docs/media/video), [`<HtmlInCanvas>`](/docs/remotion/html-in-canvas) and [`<Solid>`](/docs/solid).

## Example

```tsx twoslash title="MyComp.tsx"
import {pageTurn} from '@remotion/effects/page-turn';
import {Video} from '@remotion/media';
import {AbsoluteFill, interpolate, useCurrentFrame} from 'remotion';

export const MyComp: React.FC = () => {
  const frame = useCurrentFrame();
  const progress = interpolate(frame, [0, 45], [0, 1], {
    extrapolateLeft: 'clamp',
    extrapolateRight: 'clamp',
  });

  return (
    <AbsoluteFill>
      <Video
        src="https://remotion.media/video.mp4"
        effects={[
          pageTurn({
            progress,
            foldPosition: [0.57, 0.78],
            angle: 225,
            foldRadius: 0.12,
            lightDirection: 60,
            backOpacity: 0.85,
          }),
        ]}
      />
    </AbsoluteFill>
  );
};
```

## API

Pass an object with the following properties. You can also call `pageTurn()` without arguments.

### `progress?`

Turn progress from `0` (flat source) to `1` (fully turned away). Defaults to `0.5`.

### `foldPosition?`

Point where the page starts folding, in UV coordinates. Defaults to `[1, 1]`, the bottom-right corner.

For an edge peel, place this point on the visible edge of the source.

### `angle?`

Direction of the fold in degrees. Defaults to `225`.

Use diagonal values such as `45`, `135`, `225`, or `315` to peel from a corner.
Opposite angles peel in opposite directions.

### `foldRadius?`

Width of the curled band as a fraction of the canvas. Defaults to `0.18`.

To make a peel start exactly at an edge, use a `progress` value close to `foldRadius`.

### `lightDirection?`

Direction of the light in degrees. Defaults to `60`.

### `shadow?`

Strength of the crease and back-face shading from `0` to `1`. Defaults to `0.45`.

### `backOpacity?`

Opacity of the dimmed back face of the page from `0` to `1`. Defaults to `0.85`.

### `paperColor?`

Color of the paper back side. Defaults to `#fff8dc`.

### `disabled?`

When `true`, the effect is skipped. Defaults to `false`.

## Requirements

`pageTurn()` uses a WebGL2 backend. During renders, enable WebGL via [`Config.setChromiumOpenGlRenderer('angle')`](/docs/config#setchromiumopenglrenderer). See [Using WebGL during renders](/docs/remotion/html-in-canvas#using-webgl-during-renders).
