# outline()<AvailableFrom v="4.0.474" />

_Part of the [`@remotion/effects`](/docs/effects/api) package._

Adds an outline around the alpha edge of canvas-based components such as [`<CanvasImage>`](/docs/canvasimage), [`<Video>`](/docs/media/video), [`<HtmlInCanvas>`](/docs/remotion/html-in-canvas) and [`<Solid>`](/docs/solid).

Use it for sticker borders, silhouettes, or edge-only graphics.

## Example

```tsx twoslash title="MyComp.tsx"
import {AbsoluteFill, CanvasImage, staticFile} from 'remotion';
import {outline} from '@remotion/effects/outline';

export const MyComp: React.FC = () => {
  return (
    <AbsoluteFill>
      <CanvasImage
        src={staticFile('transparent-logo.png')}
        width={1280}
        height={720}
        effects={[
          outline({
            width: 12,
            color: '#ffffff',
            backgroundColor: '#111111',
            sourceOpacity: 0,
            displacement: 4,
            simplification: 0.25,
          }),
        ]}
      />
    </AbsoluteFill>
  );
};
```

## API

Pass an object with the following properties. You can also call `outline()` without arguments.

### `width?`

Width of the outline in pixels. Defaults to `8`. Must be at least `0`.

### `color?`

Color of the outline. Defaults to white.

### `opacity?`

Opacity of the outline from `0` to `1`. Defaults to `1`.

### `sourceOpacity?`

Opacity of the original source from `0` to `1`. Defaults to `1`.

Set it to `0` to hide the source and keep only the outline and background.

### `backgroundColor?`

Background color behind the source and outline. Defaults to transparent.

### `displacement?`

Amount of procedural edge displacement in pixels. Defaults to `0`.

### `displacementFrequency?`

Frequency of the procedural displacement. Defaults to `12`.

### `displacementSeed?`

Seed for the procedural displacement. Defaults to `0`.

### `simplification?`

Blocky edge simplification amount from `0` to `1`. Defaults to `0`.

This is a shader approximation. It snaps alpha samples to a coarse grid rather than tracing and simplifying a vector path.

### `mode?`

Which side of the alpha edge receives the outline. Defaults to `outer`.

Available values: `outer`, `inner`, `center`.

### `disabled?`

When `true`, the effect is skipped. Defaults to `false`.

## Requirements

`outline()` uses a WebGL2 backend. During renders, enable WebGL via [`Config.setChromiumOpenGlRenderer('angle')`](/docs/config#setchromiumopenglrenderer). See [Using WebGL during renders](/docs/remotion/html-in-canvas#using-webgl-during-renders).
