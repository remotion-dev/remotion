# Interactivity<AvailableFrom v="4.0.475" />

Studio interactivity means visual edits that are written back to your codebase.

Edits can be undone with <kbd>⌘/Ctrl</kbd> + <kbd>Z</kbd> and redone with <kbd>⌘/Ctrl</kbd> + <kbd>Y</kbd>.

## Selecting edit targets

Before editing, select the code-backed items that the edit should target: sequences, sequence props, effects, effect props, keyframes and easing segments.

Use <kbd>Shift</kbd> to select ranges and <kbd>⌘/Ctrl</kbd> to toggle items. Drag on the timeline to marquee-select sequences, keyframes and easing segments. Press <kbd>⌘/Ctrl</kbd> + <kbd>A</kbd> to select all sequence rows.

The outline and selection palette show which items will be affected by the next edit. Rows containing selected items receive row highlights.

## Canvas edits

Selectable sequences are outlined directly in the preview when they can be edited from the canvas.

Drag a selected outline to move the sequence by updating `style.translate`. Multiple selected sequences move together. Hold <kbd>Shift</kbd> while dragging to lock movement to one axis.

Drag outline edges to scale selected sequences when `style.scale` is editable. Drag outline corners to rotate selected sequences when `style.rotate` is editable.

Drag the transform-origin handle to move the transform origin. Studio compensates `style.translate` so the visual position stays stable.

If `style.translate`, `style.rotate` or `style.transformOrigin` is keyframed, canvas drags create or update keyframes at the current frame.

## Effects in the preview

Drop effects onto outlined sequences in the preview when the sequence supports effects.

For selected effects or effect props, drag UV-coordinate handles on the canvas to edit the effect. Connected handles are shown with lines.

Copy selected effects with <kbd>⌘/Ctrl</kbd> + <kbd>C</kbd> and paste them with <kbd>⌘/Ctrl</kbd> + <kbd>V</kbd>. You can also copy and paste individual effect prop values, including keyframed effect params where supported.

## Keyframes

Select keyframes in the timeline and drag them horizontally to change their frame.

Multiple selected keyframes move together, including keyframes for sequence props and effect props.

Press <kbd>Delete</kbd> or <kbd>Backspace</kbd> to delete selected keyframes where supported.

## Easing segments

Select easing segments between keyframes to edit interpolation.

Open the easing context menu to set `Linear`, choose a preset or open the custom easing editor. Easing edits apply to all selected easing segments.

If only easing segments are selected, press <kbd>Delete</kbd> or <kbd>Backspace</kbd> to reset them to linear.

## Deleting, resetting and duplicating

Press <kbd>Delete</kbd> or <kbd>Backspace</kbd> to delete selected keyframes, sequences, effects or all effects where supported.

When deletion does not apply, the same keys reset selected sequence or effect props to their defaults.

Press <kbd>⌘/Ctrl</kbd> + <kbd>D</kbd> to duplicate selected sequences.
