# Starburst<AvailableFrom v="4.0.435"/>

The [`<Starburst>`](/docs/starburst/starburst) component from [`@remotion/starburst`](/docs/starburst/api) renders a WebGL-based retro starburst ray effect. The effect reveals during the first half of its duration and retracts during the second half.

```tsx twoslash title="MyComp.tsx"
import {Starburst} from '@remotion/starburst';
import {AbsoluteFill} from 'remotion';

const MyComp = () => {
  return (
    <AbsoluteFill style={{backgroundColor: 'black'}}>
      <Starburst durationInFrames={60} rays={16} color1="#ffdd00" color2="#ff8800" />
    </AbsoluteFill>
  );
};
```

## Changing the number of rays

The [`rays`](/docs/starburst/starburst#rays) prop controls how many rays appear in the pattern. Values from `2` to `100` are supported.

```tsx twoslash title="ManyRays.tsx"
import {Starburst} from '@remotion/starburst';
import {AbsoluteFill} from 'remotion';

// ---cut---
const MyComp = () => {
  return (
    <AbsoluteFill style={{backgroundColor: 'black'}}>
      <Starburst rays={24} durationInFrames={30} />
    </AbsoluteFill>
  );
};
```

## Changing the colors

Use [`color1`](/docs/starburst/starburst#color1) and [`color2`](/docs/starburst/starburst#color2) to set the alternating ray colors:

```tsx twoslash title="CustomColors.tsx"
import {Starburst} from '@remotion/starburst';
import {AbsoluteFill} from 'remotion';

// ---cut---
const MyComp = () => {
  return (
    <AbsoluteFill style={{backgroundColor: 'black'}}>
      <Starburst color1="#ff0066" color2="#6600ff" durationInFrames={30} />
    </AbsoluteFill>
  );
};
```

## Rotating the pattern

Use [`rotation`](/docs/starburst/starburst#rotation) to offset the angle of the rays in degrees:

```tsx twoslash title="Rotated.tsx"
import {Starburst} from '@remotion/starburst';
import {AbsoluteFill} from 'remotion';

// ---cut---
const MyComp = () => {
  return (
    <AbsoluteFill style={{backgroundColor: 'black'}}>
      <Starburst rotation={45} durationInFrames={30} />
    </AbsoluteFill>
  );
};
```

## Using as a transition

Combined with [`<TransitionSeries.Overlay>`](/docs/transitions/transitionseries#transitionseriesoverlay), you can place a starburst at the cut point between two sequences without shortening the timeline.

```tsx twoslash title="StarburstTransition.tsx"
import {AbsoluteFill} from 'remotion';
const Fill = ({color}: {color: string}) => <AbsoluteFill style={{backgroundColor: color}} />;

// ---cut---
import {Starburst} from '@remotion/starburst';
import {TransitionSeries} from '@remotion/transitions';

const StarburstTransition = () => {
  return (
    <TransitionSeries>
      <TransitionSeries.Sequence durationInFrames={60}>
        <Fill color="#0b84f3" />
      </TransitionSeries.Sequence>
      <TransitionSeries.Overlay durationInFrames={20}>
        <Starburst rays={12} color1="#ffdd00" color2="#ff8800" />
      </TransitionSeries.Overlay>
      <TransitionSeries.Sequence durationInFrames={60}>
        <Fill color="pink" />
      </TransitionSeries.Sequence>
    </TransitionSeries>
  );
};
```

At the midpoint, the starburst covers most of the canvas, hiding the cut between scenes.
