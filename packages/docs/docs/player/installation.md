To install the player, run the following command in a React project:

<Tabs
defaultValue="npm"

{ label: 'yarn', value: 'yarn', },

<TabItem value="npm">

```bash
npm i remotion @remotion/player
```

  </TabItem>

  <TabItem value="pnpm">

```bash
pnpm i remotion @remotion/player
```

  </TabItem>
  <TabItem value="yarn">

```bash
yarn add remotion @remotion/player
```

  </TabItem>

</Tabs>
