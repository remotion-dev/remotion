Use the [`colorKey()`](/docs/effects/color-key) effect from [`@remotion/effects`](/docs/effects/api) to remove the green background of a video.

Drag the slider below to adjust how close a pixel has to be to the key color to become transparent.

<br />

```tsx twoslash title="Greenscreen.tsx"
import {colorKey} from '@remotion/effects/color-key';
import {Video} from '@remotion/media';
import React from 'react';
import {AbsoluteFill} from 'remotion';

// ---cut---
export const Greenscreen: React.FC = () => {
  return (
    <AbsoluteFill>
      <Video
        src="https://remotion.media/greenscreen.mp4"
        effects={[
          colorKey({
            similarity: 0.45,
          }),
        ]}
      />
    </AbsoluteFill>
  );
};
```

## Tuning the key

`colorKey()` accepts the following options:

- `keyColor` — the color to remove (default `'#00ff00'`).
- `similarity` — how close a pixel has to be to `keyColor` to be turned transparent.
- `smoothness` — softness of the keyed edges.
- `spillSuppression` — strength of key-color spill suppression on the remaining pixels.

See the [`colorKey()`](/docs/effects/color-key) page for the full API.

## Requirements

`colorKey()` uses WebGL2. During rendering, enable WebGL using [`Config.setChromiumOpenGlRenderer()`](/docs/config#setchromiumopenglrenderer).
