A component that renders a light leak effect as a [`<Sequence>`](/docs/sequence).

## API

The `<LightLeak>` component accepts all props that [`<Sequence>`](/docs/sequence) accepts.

```tsx twoslash
import {LightLeak} from '@remotion/light-leaks';

const MyVideo = () => {
  return (
    <LightLeak durationInFrames={60}>
      <div>My content</div>
    </LightLeak>
  );
};
```
