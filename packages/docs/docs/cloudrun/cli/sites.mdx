The `npx remotion cloudrun sites` command allows to create, view and delete Remotion projects in your Cloud Storage bucket.

## Subcommands

- [`create`](/docs/cloudrun/cli/sites/create) - Bundle and upload a Remotion video to a Cloud Storage bucket
- [`ls`](/docs/cloudrun/cli/sites/ls) - Get a list of sites in your GCP project
- [`rm`](/docs/cloudrun/cli/sites/rm) - Removes a site (or multiple) from Cloud Storage
- [`rmall`](/docs/cloudrun/cli/sites/rmall) - Remove all sites in the selected GCP project
