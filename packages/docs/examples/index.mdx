# Examples

This section will collect small, remixable motion graphics examples for Remotion.

The goal is to make examples that are easy to understand, easy to adapt, and useful as building blocks in real projects.

## Status

This section is currently a structure preview. Individual example pages will be added later.

## What belongs here?

Examples should be focused on one visual idea, technique, or workflow. They should be actionable for Remotion users rather than full, highly specific video templates.

The final example format is still being defined. A good format should work both for previewing the result and for helping users quickly use the example in their own project.

If you have an idea, see [Suggest an example](/examples/suggest-an-example).
