# @remotion/rough-notation

Rough annotation primitives for Remotion

[![NPM Downloads](https://img.shields.io/npm/dm/@remotion/rough-notation.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@remotion/rough-notation?minimal=true)

## Installation

```bash
npm install @remotion/rough-notation --save-exact
```

When installing a Remotion package, make sure to align the version of all `remotion` and `@remotion/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.remotion.dev/docs/rough-notation) for more information.
