# Athletes Eye - Remotion Template

Visualize athletic performance with GPX data on an interactive map. This template demonstrates how to create videos from GPS route data.

## Features

- Load and parse GPX files
- Display animated routes on a map
- Show athletic statistics and progress
- Smooth animations over time

## Getting Started

```bash
npm install
npm run dev
```

This will start the Remotion Studio where you can preview and compose your videos.

## Building

```bash
npm run build
```

## Rendering

```bash
npx remotion render src/index.ts <comp-id> out.mp4
```
