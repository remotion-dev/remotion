## Animated Shapes Example

Bouncing and rotating SVG shapes with spring animations.

See the full implementation in [assets/example.tsx](assets/example.tsx).

### Key Features

- Uses @remotion/shapes components (Circle, Triangle, Rect, Star)
- Staggered spring entrances for each shape
- Continuous bounce animation using sine wave
- Rotation animation for the star shape
- Stroke and fill styling
