## Typewriter with Highlight Example

Typewriter effect with blinking cursor, dramatic pause, spring-based highlight, and layer crossfade.

See the full implementation in [assets/example.tsx](assets/example.tsx).

### Key Features

- String slicing for proper typewriter effect
- Smooth blinking cursor using interpolate
- Dramatic pause between text sections
- Spring-animated highlight wipe effect
- Two-layer crossfade transition
- Configurable timing and colors
