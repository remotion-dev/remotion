## Lottie Animation Example

Loading and displaying Lottie animations with spring entrance effects.

See the full implementation in [assets/example.tsx](assets/example.tsx).

### Key Features

- Fetches Lottie JSON from remote URL
- Loading state while animation loads
- Spring-based scale and opacity entrance
- Attribution display below animation
- Uses @remotion/lottie component
