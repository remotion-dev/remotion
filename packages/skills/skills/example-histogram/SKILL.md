## Histogram Example

Animated bar chart using @remotion/shapes with staggered spring entrances and value labels.

See the full implementation in [assets/example.tsx](assets/example.tsx).

### Key Features

- Uses `@remotion/shapes` Rect component for bars
- Staggered entrance animations with spring physics
- Animated value labels that fade in
- Glow effect using CSS filter
- Responsive bar heights based on data values
