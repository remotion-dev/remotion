## Counter Animation Example

Animated number counters displaying multiple metrics with staggered entrances.

See the full implementation in [assets/example.tsx](assets/example.tsx).

### Key Features

- Multiple metric counters animating simultaneously
- Staggered entrance animations for each metric
- Support for prefixes ($) and suffixes (%)
- Tabular numeric font for stable width
- Fade and slide entrance effects
