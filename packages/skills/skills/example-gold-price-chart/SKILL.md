## Gold Price Chart Example

Elegant animated bar chart showing gold prices with Y-axis, staggered spring animations, and full-width layout.

See the full implementation in [assets/example.tsx](assets/example.tsx).

### Key Features

- Y-axis with labeled tick marks
- Staggered spring animations for each bar
- Value labels inside bars (when height is sufficient)
- Header with title and unit
- Axis lines for chart structure
- Monthly data labels
