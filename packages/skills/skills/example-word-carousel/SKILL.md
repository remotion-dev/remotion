## Word Carousel Example

Rotating words with crossfade and blur transitions.

See the full implementation in [assets/example.tsx](assets/example.tsx).

### Key Features

- Stable width container using longest word
- Smooth crossfade between words
- Blur effect during transitions
- Configurable hold and flip durations
- Prefix text with rotating suffix
