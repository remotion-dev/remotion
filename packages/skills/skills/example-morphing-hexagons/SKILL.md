## Morphing Hexagons Example

Mesmerizing hexagon grid with pulsing and color-shifting effects.

See the full implementation in [assets/example.tsx](assets/example.tsx).

### Key Features

- Procedurally generated hexagon grid
- Wave-based pulsing animation
- Dynamic HSL color shifting
- SVG glow filter effect
- Spring-based entrance animations
- Staggered delays based on grid position
