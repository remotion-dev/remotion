## Chat Messages Example

WhatsApp-style bouncy message bubbles with staggered entrances.

See the full implementation in [assets/example.tsx](assets/example.tsx).

### Key Features

- Proper chat alignment (sent right, received left)
- Spring physics for bouncy entrance
- Staggered message appearance with delays
- Scale and slide animations combined
- WhatsApp-style color scheme
